MT3000 科学上网-安装OpenClash中继一个科学节点

2023-05-04 3 min read # 软路由

MT3000 本身的系统是基于 OpenWRT 开发，进行科学上网非常方便，本身也支持中继，USB等模式，非常适合出差在酒店用。

所以如果你想拥有可以利用现有网络【手机/网线/无线】，得到一个可以无需任何设置的科学上网环境【无线有线均可以】，那么MT3000是一个比较省心的选择【嗯，439块】【理论上任何一个支持无线的 OpenWRT 的系统应该都是可以的】

配置软件源

进入 系统 - 软件包-配置OPKG，配置如下：

src/gz dllkids https://op.dllkids.xyz/packages/aarch64_cortex-a53

配置完后，点击更新列表。

如果更新失败，可以尝试多次更新，如果多次更新还是不行，建议重置固件。
