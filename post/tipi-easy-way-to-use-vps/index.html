Tipi-省心的折腾VPS服务器

2023-01-15 2 min read

闲来无事，看到搬瓦工新一代神机出炉，没忍住买了一年【😮‍💨，管不住贪便宜的手】，买都买了，要用起来啊，然后推特上看到有人推荐这玩意，试试吧。嘿，还不错，省心省力，那就用它来榨干 VPS。。。

安装后默认以80端口运行，直接输入 VPS 的 IP 地址即可打开管理面板。
当然也可以自定义下端口，毕竟 80 端口就那么一个，Tipi 安装后很绿色，就是目录下生成一个runtipi的目录，进入里面可以启动/停止/配置/更新/删除，不能更棒了。。。

➜  ~ ls
install.sh  runtipi  virt-sysprep-firstboot.log
...

当然也可以通过以下命令修改端口号：

➜  ~ cd runtipi
➜  runtipi sudo ./scripts/stop.sh

Stopping filebrowser
Starting app script
...
➜  runtipi sudo ./scripts/start.sh --port 7000

Cleaning logs folder...
Removing app.log
...
Tipi is now running
