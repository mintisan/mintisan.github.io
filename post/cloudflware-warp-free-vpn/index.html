WARP - Cloudflare 家的全平台免费 VPN

2023-04-15 3 min read # 工具使用

Cloudflare 家的免费 VPN，可以作为备份不时之需，这里记录下 WARP 如何获取 Key 来升级为 WARP+，以及申请更好用的团队 Zero Trust 版本。

【不过感觉这个羊毛很快会被国人薅成翻墙工具，然后 GFW 又将其卡之，，，】

提前准备

  • 注册 Cloudflare 账号
  • 下载 Telegram
  • 下载 WARP 客户端

WARP -> WARP+

下载安装完 WARP 后，打开直接连接：

我这边是无法使用的，网上有人反馈可以连接成功，有人反馈不行，一直停留在如下的 Connecting界面。

添加Telegram机器人@generatewarpplusbot，发送 /generate ，并计算结果即可

然后，回到 WARP ，进行 Key 的添加，一个 Key 只能被5台设备同时添加，如果不行就换一个：

等待添加成功后如下所示：

完成后，再次打开 WARP 自动转变为 WARP+，如下：

点击连接，等待成功即可。

而后可以在终端可以 Ping 通 Google，打开 SpeedTest 进行测试下，速度还可以。

但是这个需要自己去找 Key，如果后面 Key 少了，咋办呢，开个团队版本吧，，，

WARP+ -> Zero Trust Team

打开 Cloudflare 官网，点击左侧 Zero Trust 注册一个免费团队，过程中需要绑定一个信用卡或者 PyalPal，参考网站。

然后再切换到设置界面【Zero Trust-WARP Client-Device enrollment permissions manager】，设置进入 Team 所支持的邮箱后缀【注意这里不是邮箱】。

当然如果你想修改团队名称【Zero Trust-General Settings】，也是可以的。

打开 WARP 的设置界面，打开 Login to Cloudflare Zero Trust 自动弹出网页，按照网页的提示输入团队名字，然后再输入预设的后缀邮箱来进行接收授权。

等待授权成功再回来 WARP 界面自动修改为 Zero Trust 界面如下：

再打开 SpeedTest 进行测试下，速度比 WARP+ 还更快些，也用的更加的放心。

不足之处

  • IP 是会暴露的，很多人无法使用这个来进行 OpenAI 的访问
