iStoreOS - 可能是目前使用软(旁)路由的最简单姿势

2023-02-03 2 min read # 软路由

别整什么 all-in-one 了，行走江湖，但求一个【稳】字，用 iStore OS 给 R2S 弄一个专门的旁路由，还是挺香。。。

下载烧录

下载最新版本后，直接用 Balena 烧录即可。

设置旁路由模式

iStoreOS 的默认网段为 192.169.100.xxx，路由器的网段为 192.168.31.xxx，这里将 R2S 接到路由器的 LAN 口上后，再将电脑的 IP 地址修改为 192.169.100.xxx 如下：

再浏览器中输入 192.169.100.1，密码默认为 password 进入 iStoreOS

点击旁路由模式：

设置主路由的 IP 段，以及自身 IP 地址：

此时 R2S 自己会重启，我们恢复本地 IP 设置到主路由网段【注意还有DNS服务器设置为主路由192.168.31.1】：

再次在浏览器中，打开旁路由 192.169.100.2 地址：
