2023-03-12 4 min read

备份一个目录，包含 Windows/macOS/Linux/Android/iOS 上用于开发相关的软件。

All Platforms

  • Chrome : There's no place like Chrome
  • Sublime Text : Text Editing, Done Right
  • Visual Studio Code : Code editing. Redefined.
  • PyCharm : The Python IDE for Professional Developers
  • [Anaconda(https://www.anaconda.com/) : built by data scientists, for data scientists
  • CLion : A cross-platform IDE for C and C++
  • Android Studio : the official Integrated Development Environment (IDE) for Android app development.
  • Beyond Compare : allows you to quickly and easily compare your files and folders.
  • Snipaste : a simple but powerful snipping tool, and also allows you to pin the screenshot back onto the screen.
  • Git : a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
  • Docker : Develop faster. Run anywhere.
  • VNC Viewer : For the best experience install VNC® Server on the computer you want to control.
  • Filezilla : the free FTP solution
  • LANDrop : Drop any files to any devices on your LAN.
  • Paperlib : An open-source and simple academic paper management tool.
